Shell基本指令

[TOC]

  • “-“：引导短格式选项（单个字符）如：-l
  • 多个短格式选项可以写在一起，用一个”-“引导 如：-al
  • “–”：引导长格式选项（多个字符）如：–color

一、辅助快捷键

二、基本命令

1、top命令：查看系统cpu、内存等使用情况

2、date命令：系统日期设定

3、cpu信息

4、history：查看历史命令

5、关机及重启

6、man命令：查看某个命令的帮助

7、ps命令：显示进程信息

8、kill命令：干掉某个进程

三、用户命令

1、uname：（ Unix name）查看系统相关信息

2、hostname：查看系统主机名

3、useradd：添加用户账户

4、userdel：删除用户账户

5、w命令：显示登录用户的详细信息

6、last命令：查看最近那些用户登录系统

7、su命令：切换用户

8、groupadd命令：添加组

9、passwd命令：更改用户密码

10、chgrp命令：改变文件或目录的所有组

11、chmod命令：改变用户的权限

12、id命令：显示用户的信息

四、关于文件目录的命令

1、cd：（Change Directory）切换目录

2、pwd：（print work directory）查看工作目录

2、ls：（list）列表显示目录内容

3、du：（Disk usage）统计目录及文件的空间占用情况

4、mkdir：（Make Directory）创建新的目录

5、touch：新建空文件、更新时间标记

6、ln：（link）为文件或目录创建链接

7、cp：（copy）复制文件或目录

8、rm：（remove）删除文件或目录

9、mv：（move）移动文件或目录

10、find：查找文件或目录

11、进入vi：

vi三种模式：
| 模式 | 进入方式 | 操作 |
| —————– | —————– | ——————- |
| 命令模式 | ESC进入 | dd：删除 |
| | | 2dd：删除两行 |
| | | D：从光标删除到行尾 |
| | | p：粘贴 |
| | | u：恢复，返回 |
| | | “.” : 重复前一命令 |
| 插入模式 | i：光标前插入 | |
| | a：光标下一个插入 | |
| | I：行首插入 | |
| | A：行尾插入 | |
| | o：下一行插入 | |
| | O：上一行插入 | |
| 底行模式 | shift+：进入 | q!：强制退出 |
| | | wq：保存退出 |
| | wc | w：保存不退出 |
| | | set nu：显示行号 |
| ESC：回到命令模式 | | |

搜索与替换：

搜索
/string 用n搜索下一个
例: /asdf
?string 用n搜索下一个
例： ?asdf

替换

12、cat：（Concatenate 串联）显示文件的全部内容

13、wc：（word count）统计文件中的单词数量、字节数、行数

14、gzip、bzip2：压缩命令


15、tar：（Tape archive 解压文件）归档解压

16、rpm：安装、升级、卸载rpm软件包
rpm = RPM Package Manager = RedHat Package Manager

17、diff：比较目录、文件内容

18、more：分页显示命令

19、tail：显示文件的最后几行

五、网络

1、ifconfig：查看系统ip信息

网络设备
eth0 以太网
lo 回环设备
ppp0 使用ppp协议的串口设备
tr0 令牌环
fddi0 光纤2、route：查看路由表条目

3、netstat：查看网络连接情况

4、网络接口配置文件

5、域名解析配置文件

6、ping命令

7、traceroute命令：路由跟踪

8、ssh命令：远程登陆到其他UNIX主机

9、telnet命令：登陆到远程主机
